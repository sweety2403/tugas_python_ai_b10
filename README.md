Dari tugas yang telah dikerjakan, dapat disimpulkan bahwa Python memiliki berbagai fitur yang memudahkan dalam mengelola dan mengolah data. Struktur data seperti **list, tuple, set, dan dictionary** memiliki fungsi masing-masing. List digunakan untuk menyimpan dan mengubah data, tuple untuk data yang tetap, set untuk menghindari data yang sama (duplikat), dan dictionary untuk menyimpan data dengan pasangan kunci dan nilai.

Selain itu, penggunaan **comprehension** membuat penulisan kode menjadi lebih singkat dan efisien. Pengecekan data juga menjadi lebih mudah dengan menggunakan operasi keanggotaan seperti `in`.

Pada bagian fungsi dan class, kita belajar bahwa kode bisa dibuat lebih rapi dan terstruktur. **Function** membantu menjalankan tugas tertentu, sedangkan **class** membantu mengelompokkan data dan fungsi menjadi satu kesatuan, seperti pada contoh data mahasiswa.

Kemudian, dengan bantuan **NumPy dan pandas**, pengolahan data menjadi lebih cepat dan mudah, terutama untuk menghitung nilai rata-rata, median, dan membuat tabel data. Hasil pengolahan data juga bisa disimpan ke dalam file teks sehingga lebih mudah dibaca dan didokumentasikan.

Secara keseluruhan, tugas ini membantu memahami dasar-dasar Python dengan cara yang praktis dan berguna untuk kebutuhan sehari-hari, terutama dalam mengelola dan menganalisis data.
